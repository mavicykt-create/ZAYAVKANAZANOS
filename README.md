# ZAN 1.1 - Warehouse Management System

## Описание
Система управления складом/магазином для сотрудников.

## Стек технологий
- Node.js 20+
- Express
- SQLite (better-sqlite3)
- Vanilla JS + HTML + CSS
- PWA (Service Worker + Manifest)

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер:
```bash
npm start
```

3. Откройте http://localhost:3000

## Данные для входа
- admin / 7895123 (администратор)
- user / 7895123 (сотрудник)

## Модули

### 1. Заявка на занос
- Выбор товаров по категориям
- Сетка 4 товара в ряд
- Клик по карточке: +1
- Клик по кругу: -1
- Шаг 5 если в названии есть "1/..."

### 2. Проверка ценников
- Разбивка по страницам (50 товаров)
- Блокировка страниц между пользователями
- Отметка проблем и ценников

### 3. Проверка товара
- Список товаров без штрих-кода
- Возможность скрыть из списка

### 4. Календарь недели
- Просмотр событий на неделю
- Управление событиями (админ)

### 5. Админка
- Обзор системы
- Управление пользователями
- Синхронизация каталога
- Статистика

## PWA
Приложение работает как PWA:
- Установка на домашний экран
- Офлайн-режим
- Кэширование статики и изображений

## Деплой на Amvera
1. Зарегистрируйтесь на amvera.ru
2. Создайте приложение
3. Подключите GitHub репозиторий
4. Приложение автоматически задеплоится

## Синхронизация каталога
Источник: https://milku.ru/site1/export-yandex-YML/

Запуск синхронизации:
- Вручную через админку
- Автоматически (можно настроить cron)

При синхронизации:
- Загружаются товары из указанных категорий
- Изображения сжимаются до 100px WebP
- Кэш сохраняется в /data/image-cache-v5
