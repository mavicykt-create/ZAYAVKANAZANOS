# ZAN 1.1

Рабочий MVP веб-сервиса для сотрудников склада/магазина.

## Что внутри
- Node.js 20 + Express
- SQLite через better-sqlite3
- HTML/CSS/vanilla JS
- PWA: manifest + service worker + install button
- sync каталога из YML
- предсжатие изображений в `/data/image-cache-v5`
- авторизация с cookie session
- модули:
  - Заявка на занос
  - Проверка ценников
  - Проверка товара
  - Календарь недели
  - Админка
  - Статистика

## Старт
```bash
npm install
npm start
```

По умолчанию:
- admin / 7895123
- user / 7895123

## Важные пути
- База: `/data/zan.sqlite`
- Кэш картинок: `/data/image-cache-v5`

## Что реализовано в этом архиве
Это уже не пустой шаблон, а рабочий стартовый сервис:
- вход / выход
- роли admin/staff
- каталог по фиксированным категориям
- корректное отображение категорий
- carry: инкремент/декремент/подтверждение категории/сборка заявки
- price check: страницы по 50, блокировка только страниц
- product check: товары без штрих-кода, скрытие из списка
- weekly calendar CRUD для admin
- admin overview / users / sync / locks / push / stats
- sync каталога и image pre-cache

## Что стоит сделать следующим этапом
