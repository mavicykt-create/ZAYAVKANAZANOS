# ZAN 1.1 Modern

Система управления складом с современным дизайном 2025 года.

## 🎨 Дизайн-система

- **Glassmorphism** — эффекты размытия и прозрачности
- **Неоновые акценты** — cyan, purple, pink градиенты
- **Анимированный фон** — живые градиентные сетки
- **Плавные анимации** — 60fps transitions
- **Тёмная тема** — оптимизирована для работы в помещениях

## 🚀 Быстрый старт

```bash
npm install
npm start
```

Откройте http://localhost:3000

## 📱 PWA

Приложение поддерживает установку на домашний экран и работу офлайн.

## 🔧 Технологии

- Node.js 20+
- Express
- SQLite (better-sqlite3)
- Vanilla JS + Modern CSS
- Service Worker
