# ZAN 1.1 Modern - Full Deploy

Система управления складом с современным дизайном 2025 года.

## 🚀 Быстрый старт

```bash
npm install
npm start
```

Откройте http://localhost:80

## 👥 Доступ по умолчанию
- admin / 7895123 (администратор)
- user / 7895123 (сотрудник)

## 🎨 Особенности дизайна
- Glassmorphism UI с неоновыми акцентами
- Исправлено мерцание карточек на мобильных
- Фото товаров полностью вмещаются в карточки
- Плавные 60fps анимации
- Тёмная тема оптимизирована для работы в помещениях

## 📁 Структура
- server.js - главный сервер
- routes/ - API endpoints
- services/ - бизнес-логика и БД
- public/ - фронтенд (CSS, JS, HTML)
- data/ - SQLite база и кэш изображений
